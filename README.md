# ⭐ Passaparola Uygulaması

Windows Forms Application ile geliştirilmiş bilgi yarışması oyunu 🎯

---

## ✨ Proje Hakkında

Bu proje, **Windows Forms Application** kullanılarak geliştirilmiş bir **Passaparola bilgi yarışması uygulaması**dır.  
Kullanıcıların sorulara doğru cevaplar vererek ilerlediği, eğlenceli ve etkileşimli bir yarışma programı olarak tasarlanmıştır.

---

## 🚀 Özellikler

- Passaparola formatında soru–cevap sistemi
- Harf bazlı ilerleyen yarışma yapısı
- Doğru / yanlış cevap kontrolü
- Kullanıcı dostu ve sade arayüz
- Anlık geri bildirim (doğru–yanlış uyarıları)
- Masaüstü kullanımına uygun yapı

---

## 🛠 Kullanılan Teknolojiler

- C#
- Windows Forms Application
- .NET Framework
- SQL Server *(varsa, isteğe bağlı)*

---

## 🧠 Kazanımlar

Bu proje sayesinde:

- Windows Forms ile masaüstü uygulama geliştirme deneyimi kazandım
- Event tabanlı programlama mantığını pekiştirdim
- Kullanıcı etkileşimini yönetmeyi öğrendim
- Oyun mantığı ve kontrol mekanizmaları geliştirdim
